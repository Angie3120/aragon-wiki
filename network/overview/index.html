Aragon Network

The Aragon Network is the world's first digital jurisdiction, and is itself operated as a decentralized organization using Aragon.

The Aragon Network whitepaper

The Aragon Network was first described in the Aragon Network whitepaper, the first version of which was released to the public on April 14, 2017. On May 25, 2018, v2 of the Aragon Network whitepaper was released, followed by a v3 release of the whitepaper on May 10, 2019.

The Aragon Network launch

At AraCon 2019, Aragon project co-founder and then-Aragon One CEO Luis Cuende announced that the Aragon Network would launch on the Ethereum mainnet by the end of 2019. In September 2019, Aragon project co-founder and then-Aragon One CTO Jorze Izquierdo published his anticipated timeline for the Aragon Network launch, describing three main phases:

  • Phase 1: ANJ launch and Aragon Court deployment
  • Phase 2: Aragon Court launch
  • Phase 3: Aragon Network DAO

Phase 1 began with the start of Aragon Network Juror (ANJ) token pre-activation on January 7, 2020. Phase 2 to began with the start of the first precedence campaign dispute on Fight For Freedom Day February 10, 2020. Phase 3 is planned to begin when Aragon Agreements is ready for production use by the Aragon Network and other Aragon organizations.

Aragon Network governance

The Aragon Network is currently governed by a Governor Council, with plans to turn over governance to Aragon Network Token (ANT) holders in Phase 3 of the Aragon Network launch.

Aragon Network votes

The first vote ever held by the Aragon Network on the Ethereum mainnet was using the Survey app, and started on May 31, 2018. Later that year, ANT holders voted to approve AGP-1: The Aragon Governance Proposal Process. In total there were six votes held under the AGP process defined in AGP-1. On March 13, 2020 ANT holders voted to approve AGP-155, sunsetting the AGP process in preparation for Phase 3 of the Aragon Network launch.
